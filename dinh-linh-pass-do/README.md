# Đinh Linh pass đồ

## Chạy ở máy local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Đưa lên GitHub và Vercel

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ file trong thư mục này lên repository đó.
3. Vào Vercel, chọn **Add New > Project**.
4. Chọn repository GitHub vừa tạo.
5. Bấm **Deploy**.

Vercel sẽ tự nhận diện đây là project Vite.
